a) Erro: A falta da chave da direta do while.
Correção:
while( c <= 5) {
  produto *= c
  ++c;
}

b) Erro: A especificação de casas decimais no especificador de conversão.
Correção:
scanf( "%f", &valor );

if( sexo == 1 )
  printf( "Mulher\n" );
else
